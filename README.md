# Estilo bibliográfico CONICET
Podés descargar y cargar el estilo (archivo .csl) para usar en Mendeley, Zotero o cualquier gestor de bibliografías que usen.

El estilo:
* No muestra el título de los trabajos
* A partir de 3 autores ya muestra el "et al"
* Pone números en el texto, no nombres ni años
* No muestra el doi de los artículos
* Usa "y" en vez de "and" o "&" para referirse a dos autores
